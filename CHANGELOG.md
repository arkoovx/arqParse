# 📝 CHANGELOG - ArcParse

## [2.0] - 2026-04-02

### 🎯 Основное улучшение
Полный аудит и рефакторинг всего кода проекта. **2-2.5x ускорение** тестирования.

### ✨ Добавлено
- ✅ Функция `_create_session_with_retries()` в downloader.py - автоматический retry при ошибках сети
- ✅ Обработка KeyboardInterrupt во время тестирования конфигов
- ✅ Флаг `stop_flag` для четкого завершения потоков в xray_tester_simple.py
- ✅ Подсказки в консоли: "(Нажмите Ctrl+C для пропуска файла)"
- ✅ Документация: IMPROVEMENTS.md, PROJECT_STRUCTURE.md, AUDIT_REPORT.md, QUICKSTART.md

### 🔧 Изменено
- 🔄 **xray_tester_simple.py**:
  - Увеличена конкурентность: 100 → 150 потоков
  - Сокращены таймауты: 10s → 6s (timeout), 2s → 1s (port wait)
  - Увеличен check_interval: 0.01s → 0.05s (снизить нагрузку на CPU)
  - Упрощена retry стратегия (убрана задержка backoff)
  - Перемещен вывод прогресса из потока в главный цикл
  - Улучшена обработка процессов (wait=True, добавлена очистка)

- 🔄 **main.py**:
  - Удалены неиспользуемые импорты (print_progress, print_loading, type_writer)
  - Добавлена обработка Ctrl+C на разных уровнях
  - Улучшена читаемость кода

- 🔄 **parser.py**:
  - Удалены 5 неиспользуемых функций (275 строк)
  - Удалены неиспользуемые импорты (base64, unquote)
  - Версия сокращена с 360 до 85 строк

- 🔄 **downloader.py**:
  - Добавлен retry механизм (3 попытки, экспоненциальная задержка)
  - Добавлены импорты: HTTPAdapter, Retry

- 🔄 **ui.py**:
  - Удалена функция type_writer() (не используется)

- 🔄 **testers.py** и **testers_mtproto.py**:
  - Обновлены параметры по умолчанию (concurrency, timeout)
  - Улучшена синхронизация потоков

### ❌ Удалено
- 🗑️ parser.py: parse_vless_url() - дублирует xray_tester_simple._parse_vless_url()
- 🗑️ parser.py: parse_vmess_url() - не импортируется нигде
- 🗑️ parser.py: parse_trojan_url() - не импортируется нигде
- 🗑️ parser.py: parse_shadowsocks_url() - не импортируется нигде
- 🗑️ parser.py: parse_config_line() - не импортируется нигде
- 🗑️ ui.py: type_writer() - не используется в коде
- 🗑️ download_xray.py: импорт stat - не используется
- 🗑️ main.py: импорты print_progress, print_loading, type_writer

### 🟢 Исправлены баги
- 🐛 **Ctrl+C зависания**: теперь чистое завершение потоков
- 🐛 **Множественные прогресс барки**: перемещены в главный цикл
- 🐛 **Отсутствие retry сетей**: добавлены автоматические повторы
- 🐛 **Неправильная очистка процессов**: улучшена обработка SIGINT
- 🐛 **Мертвый код**: удалено 275+ строк
