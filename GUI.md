# arqParse GUI (Kivy + KivyMD)

Графический интерфейс arqParse переписан на **Kivy/KivyMD**.

## Что это даёт

- Кроссплатформенный UI: **Windows, Linux, macOS, Android, iOS**.
- Современный визуальный стиль: тёмная тема, Material Design, карточки с закруглениями.
- Плавные анимации появления элементов.

## Быстрый запуск

### Linux/macOS
```bash
./run_gui.sh
```

### Windows
```bat
run_gui.bat
```

### Прямой запуск
```bash
python main.py --gui
```

## Возможности интерфейса

- Авторизация / регистрация в один экран.
- Копирование URL подписки.
- Скачивание конфигов.
- Полный тест выбранных задач.
- Остановка активной операции.
- Просмотр live-лога и прогресса.
- Базовые настройки (в т.ч. User-Agent).

## Технические детали

- Основной GUI-файл: `gui.py`.
- Зависимости GUI: `kivy`, `kivymd` (см. `requirements.txt`).
- Переключение экранов через `ScreenManager` + `FadeTransition`.
- Операции скачивания/тестов запускаются в фоновых потоках, а UI обновляется через `Clock`.

## Примечания по мобильным платформам

Для Android/iOS обычно нужна упаковка приложения (например, через Buildozer/Xcode toolchain). Сам код интерфейса уже написан на кроссплатформенном стеке Kivy/KivyMD.
